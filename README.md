# Fastapi Starter

## Run

1. poetry install
2. make upgrade_db
3. make run
4. open http://localhost:8000/docs

## Features

- [x] Config
- [x] Logging
- [x] vscode debugger
- [x] linting
- [x] formatting
- [x] auto api docs
- [x] sql database
- [ ] migrations
- [ ] prefix ids
- [ ] cron jobs
- [ ] async/background tasks
- [ ] authentication
- [ ] authorization
- [ ] pagination
