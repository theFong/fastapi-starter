# Fastapi Starter

## Run

1. poetry install
2. `make upgrade_db env=local.env`
3. `make run env=local.env`
4. open http://localhost:8000/docs

## Features

- [x] Config
- [x] Logging
- [x] vscode debugger
- [x] linting
- [x] formatting
- [x] auto api docs
- [x] sql database
- [x] migrations
- [x] prefix ids
- [ ] pagination
- [ ] async/background tasks
- [ ] cron jobs
- [ ] authentication
- [ ] authorization
